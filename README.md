# Analisis Sentimen

Analisis sentimen sangat diperlukan dalam menyaring komentar-komentar di media sosial. Analisis sentimen pada komentar dilakukan untuk mengetahui komentar yang bersifat negatif dan positif. Analisis sentimen atau opinion mining merupakan proses memahami, mengekstrak dan mengolah data tekstual secara otomatis untuk mendapatkan informasi sentimen yang terkandung dalam suatu kalimat opini. Dalam tulisan ini analisis sentimen dilakukan untuk melihat pendapat atau kecenderungan opini terhadap sebuah masalah atau objek mengandung sentimen negatif atau positif menggunakan algoritma Na√Øve Bayes Classifier (NBC) dan TF-IDF.
